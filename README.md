# GPU Accelerated Image Processing using CUDA

## Objective
This project demonstrates GPU-based image processing using parallel computing techniques.

## Features
- Grayscale conversion
- Edge detection
- CPU vs GPU performance comparison

## Technologies Used
- Python
- CuPy
- OpenCV

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python main.py input.jpg

## GPU Usage
This project uses CuPy to perform image processing operations on GPU using parallel computation.

## Results
The GPU version processes the image faster than the CPU version.

## Files
- input.jpg → original image  
- output_gray.jpg → grayscale output  
- output_edge.jpg → edge detected output  

## Conclusion
GPU computing significantly improves performance in image processing tasks.
